Python Basic 02 函数定义

本篇文章主要介绍Python函数定义的基础知识

Jan 01, 2025
阅读时长: 2 分钟

函数定义基础概念 (◕‿◕✿)

函数是Python中可复用的代码块，通过def关键字定义：

1
2
3
4

def 函数名(参数列表):
    """文档字符串（可选）"""
    函数体
    return 返回值  # 可选
  • 命名规则：推荐使用snake_case，如calculate_average()
  • 参数：支持位置参数、默认参数等（后面会展开讲哦~）
  • 返回值：用return返回，若无return语句则返回None

举个栗子 🌰：

1
2
3
4

def greet(name):
    """输出个性化问候"""
    print(f"Hello, {name}! (ﾉ◕ヮ◕)ﾉ*:･ﾟ✧")
    return len(name)  # 返回名字长度

参数传递机制

不可变对象 vs 可变对象

  • 不可变对象（数字/字符串/元组）：传值（副本） 
    1
2
3
4

    def modify_num(x):
    x += 10  # 不影响外部变量
n = 5
modify_num(n)  # n仍是5
  • 可变对象（列表/字典）：传引用（原地修改） 
    1
2
3
4

    def append_item(lst):
    lst.append("🎉") 
my_list = [1, 2]
append_item(my_list)  # my_list变为[1, 2, "🎉"]

参数类型大全

参数类型 语法示例 说明
位置参数 func(a, b) 按顺序传参
关键字参数 func(b=2, a=1) 指定参数名传参
默认参数 def func(a=0) 参数默认值
可变位置参数 def func(*args) 接收元组 args=(1,2,3)
可变关键字参数 def func(**kwargs) 接收字典 kwargs={"x":1}

举个混合参数的例子 🌈：

1
2
3
4
5

def mixed_params(a, b=2, *args, **kwargs):
    print(f"a={a}, b={b}, args={args}, kwargs={kwargs}")

mixed_params(1, 3, 4, 5, key="value")  
# 输出: a=1, b=3, args=(4, 5), kwargs={'key': 'value'}

函数是一等公民，可以被赋值/传递 (•̀ᴗ•́)و

嵌套函数与闭包

  • 嵌套函数：函数内部定义函数

    def outer():
    def inner():
        print("我是套娃函数！(≧∇≦)ﾉ")
    return inner()
  • 闭包：携带外部作用域的变量 
    1
2
3
4

6
7

    def power_factory(exp):
    def power(base):
        return base ** exp  # 记住exp的值
    return power

square = power_factory(2)  # 专用平方函数
print(square(3))  # 输出9

装饰器（核心知识点！✨）

装饰器本质是高阶函数，用于增强函数功能：

def debug_decorator(func):
    def wrapper(*args, **kwargs):
        print(f"⚡️ 调用 {func.__name__}()")
        return func(*args, **kwargs)
    return wrapper

@debug_decorator  # 语法糖
def say_hello():
    print("Hello World!")

say_hello()  
# 输出:
# ⚡️ 调用 say_hello()
# Hello World!

函数注解（Type Hints）

Python 3.5+ 支持类型提示，用->指定返回类型：

1
2
3

def add(a: int, b: int) -> int:
    """两数相加"""
    return a + b
  • 优点：提升代码可读性，IDE能智能提示
  • 注意：不会强制类型检查（需配合mypy等工具）
