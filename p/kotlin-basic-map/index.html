Kotlin Basic Map

An introduction to Kotlin's basic grammar for Map

Jan 04, 2025
2 minute read

In Kotlin, a Map is a collection that holds pairs of objects, with each pair consisting of a key and a value. The keys in a map are unique, and each key maps to exactly one value.

Creating a Map

You can create a map using the mapOf function:

1

val map = mapOf("a" to 1, "b" to 2, "c" to 3)

This creates an immutable map. If you need a mutable map, use mutableMapOf:

1

val mutableMap = mutableMapOf("a" to 1, "b" to 2, "c" to 3)

Accessing Values

You can access values in a map using the key:

1

val value = map["a"] // value will be 1

If the key does not exist, it returns null.

Adding and Removing Entries

For mutable maps, you can add or remove entries:

1
2

mutableMap["d"] = 4 // Adds a new key-value pair
mutableMap.remove("a") // Removes the key-value pair with key "a"

Iterating Over a Map

You can iterate over the entries of a map using a for loop:

1
2
3

for ((key, value) in map) {
    println("Key: $key, Value: $value")
}

Common Operations

  • Checking if a key exists: 
    1

    val exists = map.containsKey("a") // true
  • Checking if a value exists: 
    1

    val exists = map.containsValue(1) // true
  • Getting the size of the map: 
    1

    val size = map.size // 3

Default Values

You can provide a default value if the key does not exist:

1

val value = map.getOrDefault("d", 0) // 0

Transformations

You can transform maps using various functions:

  • mapKeys: Transforms the keys of the map. 
    1

    val newMap = map.mapKeys { it.key.toUpperCase() }
  • mapValues: Transforms the values of the map. 
    1

    val newMap = map.mapValues { it.value * 2 }

Conclusion

Maps in Kotlin are versatile and provide a wide range of functionalities for storing and manipulating key-value pairs. They can be immutable or mutable, and offer various methods for accessing, modifying, and transforming the data they hold.
