Kotlin Basic for foreach and while

An introduction to Kotlin's basic grammar of for, foreach and while

Jan 06, 2025
5 minute read

for

In Kotlin, the for loop is used to iterate over a range, collection, array, or any other iterable object. It is a fundamental control flow statement that allows you to execute a block of code repeatedly.

Syntax

The basic syntax of a for loop in Kotlin is:

1
2
3

for (item in collection) {
    // action on item
}

Iterating Over a Range

You can use a for loop to iterate over a range of numbers:

1
2
3

for (i in 1..5) {
    println(i)
}

This will print numbers from 1 to 5.

Iterating Over a Collection

You can iterate over elements of a collection such as a list:

1
2
3
4

val numbers = listOf(1, 2, 3, 4, 5)
for (number in numbers) {
    println(number)
}

Iterating Over an Array

You can also iterate over elements of an array:

1
2
3
4

val array = arrayOf("a", "b", "c")
for (element in array) {
    println(element)
}

Iterating with Indices

If you need to access the index of each element, you can use the indices property of the collection:

1
2
3
4

val list = listOf("a", "b", "c")
for (index in list.indices) {
    println("Element at $index is ${list[index]}")
}

Iterating with withIndex

You can use the withIndex function to get both the index and the element:

1
2
3
4

val list = listOf("a", "b", "c")
for ((index, value) in list.withIndex()) {
    println("Element at $index is $value")
}

Iterating Over a Map

When iterating over a map, you can access both the key and the value:

1
2
3
4

val map = mapOf("a" to 1, "b" to 2, "c" to 3)
for ((key, value) in map) {
    println("Key: $key, Value: $value")
}

Nested for Loops

You can nest for loops to iterate over multi-dimensional collections:

10

val matrix = arrayOf(
    arrayOf(1, 2, 3),
    arrayOf(4, 5, 6),
    arrayOf(7, 8, 9)
)
for (row in matrix) {
    for (element in row) {
        println(element)
    }
}

foreach

In Kotlin, forEach is a higher-order function that is used to iterate over elements of a collection. It is available for various collection types such as lists, sets, and maps. Here are some key points about forEach in Kotlin:

Basic Usage:

  • forEach takes a lambda function as an argument and applies it to each element of the collection.
  • The lambda function receives each element as a parameter.

Syntax: 

1
2
3

collection.forEach { element ->
    // Do something with element
}

Example with List: 

1
2
3
4

val numbers = listOf(1, 2, 3, 4, 5)
numbers.forEach { number ->
    println(number)
}

Example with Map: 

1
2
3
4

val map = mapOf("a" to 1, "b" to 2, "c" to 3)
map.forEach { (key, value) ->
    println("$key -> $value")
}

Using forEachIndexed:

  • forEachIndexed is similar to forEach but also provides the index of each element.
1
2
3
4

val fruits = listOf("Apple", "Banana", "Cherry")
fruits.forEachIndexed { index, fruit ->
    println("Fruit at index $index is $fruit")
}

Difference from for Loop:

  • forEach is a more functional approach compared to the traditional for loop.
  • It is often preferred for its readability and conciseness.

Performance Considerations:

  • forEach might have a slight overhead compared to a for loop due to the lambda function, but the difference is usually negligible for most use cases.

Returning from forEach:

  • You cannot use return to exit from forEach as it would return from the enclosing function. Instead, you can use return@forEach to exit the lambda.
1
2
3
4
5

val numbers = listOf(1, 2, 3, 4, 5)
numbers.forEach { number ->
    if (number == 3) return@forEach
    println(number)
}

Here is a complete example demonstrating the use of forEach and forEachIndexed:

fun main() {
    val numbers = listOf(1, 2, 3, 4, 5)
    
    // Using forEach
    println("Using forEach:")
    numbers.forEach { number ->
        println(number)
    }
    
    // Using forEachIndexed
    println("\nUsing forEachIndexed:")
    numbers.forEachIndexed { index, number ->
        println("Index $index: $number")
    }
}

while

In Kotlin, the while loop is used to repeatedly execute a block of code as long as a specified condition is true. It is a fundamental control flow statement that allows you to perform repetitive tasks.

Syntax

The basic syntax of a while loop in Kotlin is:

1
2
3

while (condition) {
    // code to execute as long as condition is true
}

Example

Here is an example of using a while loop to print numbers from 1 to 5:

1
2
3
4
5

var i = 1
while (i <= 5) {
    println(i)
    i++
}

do-while Loop

Kotlin also provides a do-while loop, which is similar to the while loop but guarantees that the code block is executed at least once. The condition is checked after the code block is executed.

1
2
3
4
5

var i = 1
do {
    println(i)
    i++
} while (i <= 5)

Infinite Loop

A while loop can be used to create an infinite loop by using a condition that always evaluates to true:

1
2
3

while (true) {
    // code to execute indefinitely
}

Be cautious with infinite loops as they can cause your program to become unresponsive.

Breaking Out of a Loop

You can use the break statement to exit a while loop prematurely:

1
2
3
4
5
6
7
8

var i = 1
while (i <= 5) {
    if (i == 3) {
        break
    }
    println(i)
    i++
}

Continuing to the Next Iteration

You can use the continue statement to skip the current iteration and proceed to the next iteration of the loop:

1
2
3
4
5
6
7
8
9

var i = 1
while (i <= 5) {
    if (i == 3) {
        i++
        continue
    }
    println(i)
    i++
}
