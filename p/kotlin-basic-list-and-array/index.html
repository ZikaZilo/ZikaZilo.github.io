Kotlin Basic List and Array

An introduction to Kotlin's basic grammar for List and Array

Jan 03, 2025
3 minute read

List

In Kotlin, List is an ordered collection of elements. Here is an overview of List and its functions in Kotlin:

List Declaration

  1. Immutable List: Cannot be modified after creation.

    1

    val list: List<Int> = listOf(1, 2, 3)

  2. Mutable List: Can be modified after creation.

    1

    val mutableList: MutableList<Int> = mutableListOf(1, 2, 3)

Common List Functions

  1. size: Returns the number of elements in the list.

    1

    val size = list.size

  2. get: Returns the element at the specified index.

    1

    val element = list[0]

  3. add: Adds an element to the mutable list.

    1

    mutableList.add(4)

  4. remove: Removes the first occurrence of the specified element from the mutable list.

    1

    mutableList.remove(2)

  5. contains: Checks if the list contains the specified element.

    1

    val containsElement = list.contains(2)

  6. indexOf: Returns the index of the first occurrence of the specified element.

    1

    val index = list.indexOf(2)

  7. isEmpty / isNotEmpty: Checks if the list is empty or not.

    1
2

    val isEmpty = list.isEmpty()
val isNotEmpty = list.isNotEmpty()

  8. subList: Returns a view of the portion of this list between the specified fromIndex, inclusive, and toIndex, exclusive.

    1

    val subList = list.subList(0, 2)

Iterating Over a List

  1. Using a for loop:

    1
2
3

    for (element in list) {
    println(element)
}

  2. Using forEach:

    1
2
3

    list.forEach { element ->
    println(element)
}

fun main() {
    val list: List<Int> = listOf(1, 2, 3)
    val mutableList: MutableList<Int> = mutableListOf(1, 2, 3)

    println("List size: ${list.size}")
    println("First element: ${list[0]}")
    println("Contains 2: ${list.contains(2)}")
    println("Index of 2: ${list.indexOf(2)}")
    println("Is list empty: ${list.isEmpty()}")
    println("Sublist: ${list.subList(0, 2)}")

    mutableList.add(4)
    println("Mutable list after adding 4: $mutableList")
    mutableList.remove(2)
    println("Mutable list after removing 2: $mutableList")

    println("Iterating over list:")
    for (element in list) {
        println(element)
    }

    println("Iterating over mutable list using forEach:")
    mutableList.forEach { element ->
        println(element)
    }
}

Array

In Kotlin, an Array is a collection of elements of a specified type. Arrays in Kotlin are a fundamental data structure that allows you to store multiple values in a single variable.

Creating an Array

You can create an array using the arrayOf function:

1

val numbers = arrayOf(1, 2, 3, 4, 5)

This creates an array of integers. You can also create arrays of other types:

1

val strings = arrayOf("a", "b", "c")

Accessing Elements

You can access elements in an array using the index:

1

val firstNumber = numbers[0] // 1

Indices in arrays are zero-based, meaning the first element is at index 0.

Modifying Elements

You can modify elements in a mutable array:

1

numbers[0] = 10

Array Size

You can get the size of an array using the size property:

1

val size = numbers.size // 5

Iterating Over an Array

You can iterate over the elements of an array using a for loop:

1
2
3

for (number in numbers) {
    println(number)
}

Common Operations

  • Checking if an element exists: 
    1

    val exists = numbers.contains(3) // true
  • Finding the index of an element: 
    1

    val index = numbers.indexOf(3) // 2
  • Sorting an array: 
    1

    val sortedNumbers = numbers.sortedArray()

Creating Arrays of Primitive Types

Kotlin provides specialized classes for arrays of primitive types to avoid the overhead of boxing:

  • IntArray
  • DoubleArray
  • CharArray
  • BooleanArray
  • etc.

Example:

1

val intArray = intArrayOf(1, 2, 3, 4, 5)

Initializing Arrays

You can initialize arrays with a specific size and a lambda function to generate the elements:

1

val squares = Array(5) { i -> i * i } // [0, 1, 4, 9, 16]
