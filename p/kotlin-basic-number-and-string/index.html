Kotlin Basic Number and String

An introduction to Kotlin's basic grammar for Number and String

Jan 02, 2025
5 minute read

Number

In Kotlin, number types include Byte, Short, Int, Long, Float, and Double. Here is an introduction to Kotlin’s basic grammar for number types and their functions:

Number Type Declaration

You can declare number variables using val or var:

1
2
3
4
5
6

val byteVal: Byte = 1
val shortVal: Short = 10
val intVal: Int = 100
val longVal: Long = 1000L
val floatVal: Float = 10.5F
val doubleVal: Double = 20.5

Type Inference

Kotlin can infer the type of the variable from the assigned value:

1
2
3
4
5
6

val byteVal = 1.toByte()
val shortVal = 10.toShort()
val intVal = 100
val longVal = 1000L
val floatVal = 10.5F
val doubleVal = 20.5

Common Number Functions

  1. toByte: Converts the number to Byte.

    1

    val byteVal = intVal.toByte()

  2. toShort: Converts the number to Short.

    1

    val shortVal = intVal.toShort()

  3. toInt: Converts the number to Int.

    1

    val intVal = longVal.toInt()

  4. toLong: Converts the number to Long.

    1

    val longVal = intVal.toLong()

  5. toFloat: Converts the number to Float.

    1

    val floatVal = doubleVal.toFloat()

  6. toDouble: Converts the number to Double.

    1

    val doubleVal = floatVal.toDouble()

Arithmetic Operations

Kotlin supports standard arithmetic operations:

1
2
3
4
5

val sum = intVal + longVal
val difference = doubleVal - floatVal
val product = intVal * shortVal
val quotient = doubleVal / floatVal
val remainder = intVal % shortVal

Comparison Operations

Kotlin supports standard comparison operations:

1
2
3
4
5
6

val isEqual = intVal == longVal.toInt()
val isNotEqual = intVal != longVal.toInt()
val isGreater = doubleVal > floatVal
val isLess = intVal < longVal
val isGreaterOrEqual = doubleVal >= floatVal
val isLessOrEqual = intVal <= longVal

fun main() {
    val intVal = 100
    val longVal = 1000L
    val floatVal = 10.5F
    val doubleVal = 20.5

    println("Int to Long: ${intVal.toLong()}")
    println("Long to Int: ${longVal.toInt()}")
    println("Float to Double: ${floatVal.toDouble()}")
    println("Double to Float: ${doubleVal.toFloat()}")

    println("Sum: ${intVal + longVal}")
    println("Difference: ${doubleVal - floatVal}")
    println("Product: ${intVal * 2}")
    println("Quotient: ${doubleVal / floatVal}")
    println("Remainder: ${intVal % 3}")

    println("Is Equal: ${intVal == longVal.toInt()}")
    println("Is Not Equal: ${intVal != longVal.toInt()}")
    println("Is Greater: ${doubleVal > floatVal}")
    println("Is Less: ${intVal < longVal}")
    println("Is Greater Or Equal: ${doubleVal >= floatVal}")
    println("Is Less Or Equal: ${intVal <= longVal}")
}

String

In Kotlin, String is a class that represents a sequence of characters. It is immutable, meaning that once a String object is created, it cannot be changed. Here is an overview of String and its functions in Kotlin:

String Declaration

You can declare a string using double quotes:

1

val str: String = "Hello, World!"

String Templates

Kotlin supports string templates, which allow you to embed variables and expressions inside a string:

1
2
3

val name = "John"
val greeting = "Hello, $name!"
val lengthMessage = "The length of the name is ${name.length}"

Common String Functions

  1. length: Returns the length of the string.

    1

    val length = str.length

  2. substring: Returns a substring from the specified start index to the end index.

    1

    val subStr = str.substring(0, 5)  // "Hello"

  3. toUpperCase / toLowerCase: Converts the string to upper case or lower case.

    1
2

    val upper = str.upperCase()
val lower = str.lowerCase()

  4. trim: Removes leading and trailing whitespace.

    1

    val trimmed = str.trim()

  5. split: Splits the string into a list of substrings based on the specified delimiter.

    1

    val parts = str.split(", ")

  6. replace: Replaces occurrences of a substring with another substring.

    1

    val replaced = str.replace("World", "Kotlin")

  7. contains: Checks if the string contains the specified substring.

    1

    val containsHello = str.contains("Hello")

  8. startsWith / endsWith: Checks if the string starts or ends with the specified prefix or suffix.

    1
2

    val startsWithHello = str.startsWith("Hello")
val endsWithWorld = str.endsWith("World!")

  9. indexOf / lastIndexOf: Returns the index of the first or last occurrence of the specified substring.

    1
2

    val index = str.indexOf("World")
val lastIndex = str.lastIndexOf("o")

  10. isEmpty / isNotEmpty: Checks if the string is empty or not.

    1
2

    val isEmpty = str.isEmpty()
val isNotEmpty = str.isNotEmpty()

String Iteration

You can iterate over the characters of a string using a for loop:

1
2
3

for (char in str) {
    println(char)
}

Raw Strings

Kotlin supports raw strings, which can contain newlines and arbitrary text. They are enclosed in triple quotes ("""):

1
2
3
4

val rawString = """
    This is a raw string.
    It can span multiple lines.
    """

String Equality

In Kotlin, you can check string equality using the == operator, which checks for structural equality:

1
2
3

val str1 = "Hello"
val str2 = "Hello"
val isEqual = str1 == str2  // true

fun main() {
    val str = "Hello, World!"
    println("Length: ${str.length}")
    println("Substring: ${str.substring(0, 5)}")
    println("Upper case: ${str.upperCase()}")
    println("Lower case: ${str.lowerCase()}")
    println("Trimmed: ${str.trim()}")
    println("Split: ${str.split(", ")}")
    println("Replaced: ${str.replace("World", "Kotlin")}")
    println("Contains 'Hello': ${str.contains("Hello")}")
    println("Starts with 'Hello': ${str.startsWith("Hello")}")
    println("Ends with 'World!': ${str.endsWith("World!")}")
    println("Index of 'World': ${str.indexOf("World")}")
    println("Last index of 'o': ${str.lastIndexOf("o")}")
    println("Is empty: ${str.isEmpty()}")
    println("Is not empty: ${str.isNotEmpty()}")
}
